Mini-sudoku project

Mini-sudoku is a smaller version of the full-size 9x9 sudoku.

Mini sudoku has a 4x4 grid with 4x4 squares.

The purpose of the project was to create a functional mini-sudoku game.

My part in the project was to make the functionality of the score table for other features as well.

I was the equipment manager in the project. I acquired the necessary components for the project, such as theLCD screen and the buttons.

Banksimul project

The ATM simulator is a computer-based program through which you can

to simulate the operation of an ATM, which is intended to be very close to the real ATM operation.

The card that was used in the project was an RFID card that had a corresponding RFID reader.

My part in the project was to make the functionality of the ATM, such as withdrawing money, checking the balance, and reading the bank card.

I did most of the backend myself.

The project was challenging, but very fun to do.

Film Club project

The Film Club project or (Elokuvakerho) was a project where we created a website using react, where you can search for movies and see their information.

The project used an external database where the information of the movies / series was read from. In the project we also created a simple database of our own,

where we stored the information of the users who had registered, which user liked which movie and more on the site.

My part of the project was to create and modify the backend for our custom stuff we wanted to add to our website. I also was part of creating and modifying the frontend when new stuff / content was added

Good Day project

The Good Day project was a project where we created a website/application using react native, where the emphasis was to make a application for a mobile device.

The program was made with people in mind that need these sort of applications, where there are relatively simple but important features such as to-do lists, calendar where you can make your own markings to, weather and a simple entertainment pages.

My part of the project was to make the weather page and the weather 'widget' for the home screen.
